How was this project made?

This project was made with HTML, CSS and Javascript. Check out the code at www.github.com/Krishna-Sivakumar/browse_westeros

Implementation Details Below

The map above has legends that get zoomed on when hoevered over, which was implemented with a canvas. The canvas has the map image as a background, and draws a clipped and enlarged version of the source image around a legend if the cursor is over it.

Every time the cursor moves, an event is triggered to check whether the mouse is over a legend[ 1 ]. If so, a zooomed section is drawn, or else the drawn highlight is removed. Clicking on a legend also triggers an onclick method that navigates to a page in the Game of Thrones wiki.
