How was this project made?

This project was made with HTML, CSS and Javascript.

Implementation Details Below

The map above has legends that get highlighted when hoevered over, which was implemented with a canvas. The canvas has the map image as a background, and draws a low-alpha rectangle around a legend if the cursor is over it.

Every time the cursor moves, an event is triggered to check whether the mouse is over a legend. If so, a highlight is drawn, or else the drawn highlight is removed. Clicking on a legend also triggers an onclick method that navigates to a page in the Game of Thrones wiki.
